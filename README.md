# NeonCrush - Cyberpunk Image Optimizer

High-performance, client-side image compression tool with a premium cyberpunk aesthetic.

## Features
- **Privacy First**: 100% Client-side processing. Images never leave your browser.
- **Advanced SVG Support**:
    - **Minify**: Clean up SVG code.
    - **Rasterize**: Convert SVG to PNG/JPG.
    - **GIF Conversion**: Turn SVG animations into GIFs with **Auto-Detect Duration** and **Smart Auto-Compression**.
- **Smart Compression**:
    - **Quality Slider**: Fine-tune lossy compression (1-100%).
    - **Manual Control**: Queue images and compress when ready.
    - **Comparison**: Visual Before/After slider to verify quality.
- **Formats**: JPG, PNG, WebP, SVG, GIF.
- **Batch Processing**: Process up to 60 images at once.
- **Cyberpunk UI**: Premium dark mode design with glassmorphism.

## Usage
1. Open `index.html` in your browser.
2. Drag & Drop images.
3. For SVGs, choose your optimization method (Minify/Raster/GIF).
4. For Images, adjust quality and click "Compress Now".
5. Click the "Compare" icon to check results.
